# CS-322 Assignment 2
### This project focuses on working with the ASM Bytecode library in java. There are eight classes that run simple bytecode programs.
- MInt multiplies two integers, and multiplies two doubles. The integers and doubles are already declared in the program, no user input required.
- SLo subtracts two longs, and subtracts two doubles. The longs and doubles are already declared, no user input.
- DFlo divides two floats, then divides two doubles. The floats and doubles are already declared, no user input.
- PrGT compares two integers, and prints the larger one. Both integers are already declared, no user input.
- PrStr prints two strings. Strings are already declared, no user input.
- IfElse compares a declared number to 0, and prints if it's equal to zero or not. The declared number in the program is currently 0, so it will print that the number is equal to zero.
- Echo takes a string input from a user, and prints back the same string. String input from user on command line is required.
- NegTest takes an integer input from a user, and prints whether it is negative or non-negative. Integer user input is required.
